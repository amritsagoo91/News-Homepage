Creative Commons Image Tool

When searching for images on Google Images if you want something with the creative commons licence then you can go to the tools section on the far right of the menu bar and select the usage rights dropdown. From there you can select the creative commons licence and it will filter out all images that do not have that licence. This is a useful tool for finding images that can be used for free, but it is important to always check the specific licence for each image to ensure that you are using it correctly and legally.
